# NumPy Assignments

Practice assignments covering core NumPy concepts.

## Topics Covered
- Array basics, indexing & slicing
- 2D arrays & matrix operations
- Random numbers & statistics
- Mathematical & conditional operations
- Reshaping & stacking
- Sorting & searching
- Linear algebra
- Mini data analysis

## Requirements
pip install numpy

## Usage
Run any file individually:
python q1_numpy_basics.py
